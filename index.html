☁️ Pengaturan Cloud Sync
Cara setup (gratis, 2 menit):
1. Buka jsonbin.io → daftar gratis
2. Dashboard → klik + Create Bin → isi [] → Save
3. Copy BIN ID (angka panjang di URL)
4. Klik Account → API Keys → copy Master Key
5. Tempel di bawah → Simpan
Temukan di: Account → API Keys → Master Key
Temukan di: URL bin kamu, contoh: jsonbin.io/v3/b/INI-BIN-ID-NYA
📥 Import dari CSV
Kolom wajib: date, amount
Kolom opsional: platform, product, nama, email, hp, status
Format tanggal: YYYY-MM-DD atau DD/MM/YYYY
Status: completed atau pending
